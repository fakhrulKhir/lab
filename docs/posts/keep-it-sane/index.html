Keep It Sane

Jun 16, 2019 03:34 ¬∑ 401 words ¬∑ 2 minute read workflow xp

This “Keep It Sane” list is a lisf of problems I find myself recurringly trying to figure solution for. Not surprisingly, the solution often came from the page or blog post that I’ve read before.

The list would serve as a shortcut that saves me time from trying to google the solution again and again. It’s like index of things that had bitten me in the past, which I don’t want to get bitten by again.

macOS

collapse

  1. Source Tree keep asking for password:
    • tldr: git config --global credential.helper osxkeychain
    • ref: stackoverflow
  2. Keychain asking password two times
    • (Not solved yet)
  3. NAS disk prompt keeps popping up
    • (Not solved yet)
  4. Activate “Do Not Disturb” mode (disable all notifications)
    • alt+click on notification icon in menu bar. See: macOS DnD.
  5. macOS won’t boot. Question mark (with disk icon behind it) appears during boot
    • Unfortunately, your SSD might have failed you. Try replacing it.
    • Blog post: WIP

Ubuntu

collapse

  1. Stuck in boot screen (Ubuntu’s purple screen)
    • Login to recovery mode, run fsck, reboot.
  2. Recover data from unbootable ubuntu
    • Boot to Finnix rescue cd, activate ssh server, recover data using sftp. See: Vultr blog post (or its snapshot).

Android

collapse

  1. Termux can’t access device storage
    • Open termux and run termux-setup-storage
  2. Termux history is not active
    • (Not solved yet)
  3. Share termux shell as web app
    • Download gotty, run ./gotty -w sh, and visit http://<device-ip>:8080 from browser.

Dev

collapse

  1. Debug GraphQL requests in browser
    • Use GraphQL Network extension

  2. GraphQL error “GROUP BY clause is incompatible with sql_mode=only_full_group_by”

    • Remove ‘ONLY_FULL_GROUP_BY’ from mysql’s sql_mode. Example:

      SET GLOBAL sql_mode=(SELECT REPLACE(@@sql_mode,'ONLY_FULL_GROUP_BY,',''));

    • See: stackoverflow

Workflow

collapse

  1. Debugging NodeJS, Go, Bash, etc. with vscode
    • (Blog post in progress)
  2. Need to quickly switch proxy server for command line environment
    • use proxify
  3. Preview JSON response in Opera:
    • Install json-lite extension
  4. need to grab screen shot and use it as quick reference
    • use grabit
  5. need to upload file and share it randomly
    • use transfer.sh
  6. SublimeText 3: show current file in sidebar
    • ctrl+0 (ctrl+zero). See: stackoverflow
  7. Display full path of file in sublime
    • See: stackoverflow
  8. change command+click handler in iterm (macOS)
    • (Not solved yet)
  9. Save time typing localhost by aliasing l to localhost
    • echo '127.0.0.1 l' >> /etc/hosts. curl l:3000 is now equivalent to curl localhost:3000
  10. Convenient scripts:
    • collate: combine images to one pdf file
    • makegif, makemp4: convert videos into (optimized) gif/mp4
